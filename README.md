# Прогнозирование оттока клиентов на основе RFM-метрик

### Решаемая проблема:

Маркетологам не виден момент ухода клиента. Это приводит к лишним тратам на лояльных покупателей и потере тех, кого можно было удержать.

### Решение:

1. На основе RFM-метрик обучена модель прогнозирования оттока 
2. Сформирован список с указанием вероятности оттока и выручки по каждому клиенту

### Инструменты:

Pandas, Scikit-learn, Matplotlib/Seaborn

### Результат:

Маркетологи видят вероятность оттока и ценность клиентов. Бюджет можно перенаправить на тех, кто приносит больше прибыли и имеет более высокий риск оттока.
